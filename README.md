# hastebin-bot
An opensource Discord bot that posts data to Hastebin.
Written in Node.js

**Dependencies**

-Node.js

-Discord.js

-Hastebin-Gen

**How to use**

Add your token into the config.json fle, and then add commmands and stuff, then run the file in a node command prompt or something.
